## 🌈 项目简介

> **MBTI 已经过时，SBTI 来了！**

SBTI 是一套完全基于当代年轻人精神状态构建的人格分析系统。它抛弃了传统心理学框架的束缚，真正深入到每一个抽象灵魂的内核。

- 🎯 **15 个分析切面** = 5 大模型 × 3 维度
- 📊 **27 种人格类型** 通过 31 道精心设计的题目推导
- 😂 **不关心你是内向还是外向**，只关心：你到底是谁？你在过一种怎样的人生？
